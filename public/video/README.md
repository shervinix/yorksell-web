# Hero background video

Place a Toronto skyview (or any aerial city) video here as **hero.mp4** to use it as the home page hero background.

- **Recommended:** MP4, 1080p or 4K, 10–30 seconds, muted (no audio needed).
- **Where to get one:** Free options include [Coverr](https://coverr.co/stock-video-footage/toronto), [Pexels](https://www.pexels.com/search/videos/toronto%20skyline/), or [Mixkit](https://mixkit.co/free-stock-video/skyline/). Download and save as `hero.mp4` in this folder.

If `hero.mp4` is not present, the hero will show the static poster image instead.
